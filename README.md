# Prime Development Kit (PDK)

> Metadata-driven application development platform for .NET 10, FastEndpoints and Vertical Slice Architecture.

---

## What is PDK?

Prime Development Kit (PDK) is an open-source framework that enables developers to define application structure using metadata and generate repetitive application artifacts automatically.

PDK focuses on:

* Vertical Slice Architecture (VSA)
* FastEndpoints
* Entity Framework Core
* Metadata-driven development
* Code generation
* Developer-owned business logic

---

## Key Principles

* Metadata owns structure and intent.
* Code owns behavior.
* Entities remain the source of truth.
* Feature Profiles define application capabilities.
* Generators produce infrastructure code.
* Developers retain full ownership of business logic.

---

## ⚡ Powered By

PDK links together an elite group of performance-first .NET libraries to deliver lightweight vertical slices:
* **[FastEndpoints](https://fast-endpoints.com/)** — Drives the REPR (Request-Endpoint-Response) Minimal API layer and provides the high-performance internal native Command Bus.
* **[QueryKit](https://github.com/pdevito3/QueryKit)** — Evaluates complex data parsing, filtering, and sorting safely over client-driven DTO query parameters.
* **[Riok.Mapperly](https://mapperly.riok.app/)** — Generates zero-allocation, compile-time projection mapping code instantly for CRUD-bound slices.
